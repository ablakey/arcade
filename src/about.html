About

The Arcade is a simple game engine and small library of games written by Andrew Blakey

Controls

WASD/Arrows + SPACEBAR or on-screen buttons. Not every button is used in every game.

Philosophy

I've spent years trying to make games as a relaxing hobby. Unfortunately, most usually get too complicated or balloon in scope.

The goal of this project is to have an intentionally primitive game engine that allows me to write a game in about a day.

Technical Details

For an FAQ, see the README on GitHub. PRs for fixes or new games are welcome. Feel free to fork and use it as your own.
